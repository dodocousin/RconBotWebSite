Features

Advanced Game Server Management Solution RCON Integration: Utilize precise slash commands to establish connections with multiple game servers directly through Discord.

Enhanced Security Protocols: Server credentials are securely encrypted and stored, ensuring protected access and maintaining data integrity.

Comprehensive Server Control: Execute RCON commands with immediate effect, providing full administrative capabilities over your game servers within the Discord environment.

Robust User Authentication: Implement our advanced player linking system to enforce strict access control, allowing only authorized personnel to interact with your servers.

High-Performance Architecture: Built on the .NET 8 framework, this bot delivers exceptional speed and reliability in command execution and response.

Flexible Configuration: Adapt the bot to your specific requirements with customizable command prefixes and server-specific settings.

Persistent Connectivity: Benefit from automatic reconnection protocols, ensuring consistent uptime and uninterrupted service delivery.

Multi-Server Compatibility: Efficiently manage a diverse array of game servers across various platforms from a single, centralized interface.

Streamlined Deployment: Experience a straightforward setup process, allowing for quick integration and immediate operational capability.

Empowering administrators with professional-grade tools for optimal game server management and control.

What the bot can do

Permission management

Select the discord roles that can access administrative commands

Example: Use the /permission_add to add a role to use admin commands

Cluster management

Group servers into clusters for easier management and execute commands across multiple servers simultaneously. Example: Use the /cluster_rcon to update all servers in a cluster at once.

Player linking

Link player game accounts with their Discord accounts for better player management and interaction.

Example: Use the /link command to link your game account with your Discord account.

Player management

List and show players informations
